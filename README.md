# Portfolio - Vite + TypeScript

This project was rebuilt from CRA to Vite with TypeScript.

## Scripts

1. `npm run dev` - start local development server.
2. `npm run start` - alias to `npm run dev`.
3. `npm run build` - run TypeScript check and create production build in `dist`.
4. `npm run preview` - serve built app locally.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```
